Программа предназначена для проведения побайтовых операций над файлами в указанной директории, 
в том числе и в поддиректорях, при установке соответствующего флага. 

Для запуска кодирования необходимо указать исходную и выходную папку, а также маску файлов, например:
*.txt
*.txt | *.pdf

В случае, если не указан ключ шифрования, то фалы будут просто скопированы в выходную директорию. 
Максимальная длинна ключа - 32 символа.

Возможен запуск по таймеру, с минимальным интервалом 1 минута. Указывается в формате: hh:mm.
При работе по таймеру интерфейс блокируется, для остановки таймера и разблокировки интерфейса необходимо нажать кнопку "Остановить".

Если входная директория содержит путь выходной директории, то копирование содержащихся в исходной папке файлов не происходит.

Возможна перезапись файлов при совпадении имён при указании соответствующего флага, а также удаление исходных файлов.
